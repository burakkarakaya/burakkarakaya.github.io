# Çalışma Şekli

1. Önce design klasöründeki referansları analiz et.
2. Sonra docs klasöründeki logo ve ürün ambalajlarını incele.
3. Ardından kısa bir tasarım yaklaşımı yaz.
4. Son olarak HTML + CSS kodunu üret.

Referansları incelemeden tasarıma başlama.

Amaç sadece güzel görünmek değil.
